# Home Improvement Club Website

Marketing website for Home Improvement Club, built with React + TypeScript + Vite.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Framer Motion

## Getting Started

### Prerequisites

- Node.js 18+ (recommended)
- npm

### Install

```bash
npm install
```

### Environment Variables

Create a `.env.local` (or `.env`) file in the project root:

```env
FORMSPREE_ID=your_formspree_form_id
```

## Development

Run the local dev server:

```bash
npm run dev
```

Default dev server: `http://localhost:8080`

## Build and Preview

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Testing and Linting

Run lint checks:

```bash
npm run lint
```

Run tests once:

```bash
npm run test
```

Run tests in watch mode:

```bash
npm run test:watch
```
